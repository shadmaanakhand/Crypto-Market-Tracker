# Crypto Dashboard

A real-time cryptocurrency dashboard built with React, Tailwind CSS, and the CoinGecko API.
It displays live market data, interactive price charts, and allows users to search and track cryptocurrencies in a clean, responsive interface.

## Features

* Live prices for the top 100 cryptocurrencies

* Auto-refreshes market data every 60 seconds

* 24h price change with green/red indicators

* Search across CoinGeckoâ€™s full crypto database

* Interactive price charts (7d / 30d / 90d)

* Compare two cryptocurrencies side by side

* Favorites / watchlist (saved in local storage)

* Dark & light mode

* Sortable table columns (price, market cap, volume, % change)

* Fully responsive (mobile, tablet, desktop)

## Tech Stack
* React 18
* Tailwind CSS
* CoinGecko API (no API Key required)
* Recharts
* Lucide Reacts

## Deployment
This is a fully client-side app and can be deployed to:

* Netlify
* GitHub Pages
